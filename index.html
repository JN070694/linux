Debian

  • System Settings:
  • (Configure taskbar to left pane)
  • System Settings > Accessibility > Screen Reader: Uncheck
  • System Settings > Color & Themes > Global Theme > Breeze Dark
  • System Settings > Color & Themes > Login Screen (SDDM)
  • System Settings > Power Management > 5,5,1
  • Disable WiFi PowerSave
  • Update NVIDIA GPU Driver

  • Setup Commands:
  • sudo apt purge akregator
  • sudo apt purge firefox-esr
  • sudo apt purge konqueror
  • sudo apt purge plasma-welcome
  • sudo apt install curl
  • sudo apt install fastfetch
  • sudo apt install git
  • sudo apt install maliit-keyboard

  • To Install Preferenced Packages
  • curl -fsSL https://agelesslinux.org/become-ageless.sh | sudo bash -s -- --accept --flagrant
  • LibreWolf, 3 Commands
  • sudo apt update && sudo apt install extrepo -y
  • sudo extrepo enable librewolf && sudo extrepo update librewolf
  • sudo apt update && sudo apt install librewolf -y
  • Required Add-on: Bitwarden

  • To Install Flatpaks, (KDE)
  • sudo apt install flatpak
  • sudo apt install plasma-discover-backend-flatpak
  • flatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo

  • To Install Preferred Flatpak Packages
  • flatpak install flathub eu.betterbird.Betterbird
  • flatpak install flathub com.bitwarden.desktop
  • flatpak install flathub com.discordapp.Discord

  • Install .deb Packages
  • (Download .deb package, like Steam's for example)
  • cd Downloads
  • sudo apt install ./steam_latest.deb
  • sudo apt install ./

  • Update PC Sequence
  • sudo fwupdmgr refresh && sudo fwupdmgr update -y && \
  • sudo apt update && sudo apt full-upgrade -y && \
  • sudo apt autoremove --purge -y && sudo apt autoclean && \
  • flatpak update -y && flatpak uninstall --unused -y && \
  • for dir in ~/.var/app/*/; do [ -d "$dir" ] && app=$(basename "$dir") && ! flatpak info "$app" &>/dev/null && { echo "Removing orphaned: $app"; rm -rf "$dir"; }; done
